Site Entrance

Welcome to my_site

Open the public blog, browse media libraries, manage your profile, and move into the platform's current gallery, album, audio, and protected video flows from one entrance page.

Blog

Open the public homepage, browse posts, search content, and jump into profile-linked author pages and RSS-driven publishing flows.

Accounts and Profile

Go directly to login, registration, or the profile page with paginated posts, comments, gallery uploads, albums, audio, video, and avatar management.

Create and Upload

Jump to the common authenticated actions: create a post, upload audio, upload a gallery image, or create an album batch.

Media Libraries

Open the current public media sections for audio, video, gallery images, and albums. Media delivery is routed through proxy endpoints instead of exposing storage paths in the normal UI.

Current stack: Django, PostgreSQL, Redis, Elasticsearch, Celery, Docker Compose, and optional Flower / Prometheus / Grafana services. This portal now matches the routes and media features currently active in the project.
